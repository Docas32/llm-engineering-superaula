# LLM Engineering, RAG & AI Agents Masterclass 

Este repositório é dedicado ao registro da minha evolução técnica durante a Masterclass de Engenharia de LLMs. Como engenheiro de formação e em transição para a área de dados, utilizo este espaço para aplicar conceitos teóricos em soluções práticas e escaláveis.

## Objetivo do Repositório
Documentar o aprendizado em arquiteturas de Inteligência Artificial Generativa, focando em:
* **RAG (Retrieval-Augmented Generation):** Otimização de busca e recuperação de contextos.
* **AI Agents:** Construção de agentes autônomos com capacidade de raciocínio e ação (ReAct).
* **LLM Engineering:** Melhores práticas de integração, segurança e custo-performance.

## Stack Tecnológica (Módulo 1)
* **Linguagem:** Python 3.12
* **Modelo:** Google Gemini 2.5 Flash (via Google GenAI SDK v1.0)
* **Ambiente:** Pop!_OS Linux
* **Segurança:** Gestão de chaves via `python-dotenv`.

## Destaques de Engenharia Aplicada
No primeiro módulo, foquei na migração de bibliotecas legadas para o SDK mais recente do Google, garantindo o uso do modelo **Gemini 2.5 Flash**. Além disso, implementei um sistema de **System Instructions** modularizado através de um dicionário de personalidades, permitindo que a IA assuma papéis específicos, como:

* **Mentor Técnico:** Focado em arquitetura e código limpo.
* **Especialista em Licitações:** Conexão direta com meu projeto pessoal **Licita-Hunter**, aplicando IA para analisar editais e documentos de engenharia.
